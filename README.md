# Navigator | The Universal AI Guide

Navigator is an intelligent overlay that guides users through any SaaS tool (Notion, GitHub, etc.) with step-by-step instructions generated by AI.

## ðŸ“‚ Project Structure

This monorepo contains the entire MVP stack:

- **`/extension`**: The Chrome Extension.
  - Injects the guidance overlay into target websites.
  - Captures screenshots (eyes) and handles user interaction.
  - Contains autonomous "Embedded Brain" for offline resilience.

- **`/backend`**: Convex Database & API.
  - Stores execution history, knowledge base, and tool definitions.
  - Handles real-time synchronization between Extension and n8n.

- **`/n8n`**: The AI Brain (Workflows).
  - `04 - Tool Detection`: Identifies what app the user is looking at.
  - `06 - Generate Guidance`: Analyzes screen + query to create steps.
  - `05 - Next Step`: validation and progress tracking.

- **`/frontend`**: Web Dashboard.
  - Admin interface to view active sessions, knowledge, and analytics.
